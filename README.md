Modern authentication forces a trade-off between security and user privacy. **Praman Network** changes this paradigm by introducing Identity-as-a-Service (IaaS) powered by Zero-Knowledge Proofs (ZK-SNARKs). We enable dApps to verify users through biometrics *without ever transmitting or storing raw biometric data*.

## 🚀 What We Are Building

- 📦 **[PramanAuth SDK](https://www.npmjs.com/package/@praman-network/sdk):** Our flagship NPM package. A plug-and-play solution to integrate gasless, passwordless biometric auth into any React/Next.js dApp.
- ⚡ **Decentralized Relayers:** A highly optimized backend infrastructure that sponsors gas fees and submits verifiable ZK-proofs to the Polygon network.
- 🔐 **On-Chain Registry:** Smart contracts deployed on Polygon (Amoy) that maintain a sybil-resistant registry of verified human identities.

## 🛠️ Our Tech Stack

- **Cryptography:** SnarkJS, Groth16, Zero-Knowledge Circuits
- **Blockchain:** Solidity, Polygon, Ethers.js
- **Infrastructure:** Node.js, Express.js, IPFS
- **Frontend Ecosystem:** React, Next.js, WebAssembly (WASM)
